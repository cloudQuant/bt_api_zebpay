# bt_api_zebpay

Zebpay exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_zebpay
```

## Usage

```python
from bt_api_zebpay import ZebpayExchangeDataSpot, ZebpayRequestDataSpot
```

## Features

- REST API support
- HMAC-SHA256 authentication
- Symbol format: `BTC-INR`
